AxisTables

Construction

Matrix to Table

julia> using AxisTables

julia> t = Table(ones(2,2), col_keys=[:A, :B])
Table
┌─────┬─────┐
│   A │   B │
├─────┼─────┤
│ 1.0 │ 1.0 │
│ 1.0 │ 1.0 │
└─────┴─────┘

Key Word Construction

julia> using AxisTables

julia> t = Table(A = 1:4, B = ["M", "F", "F", "M"])
Table
┌───┬───┐
│ A │ B │
├───┼───┤
│ 1 │ M │
│ 2 │ F │
│ 3 │ F │
│ 4 │ M │
└───┴───┘
